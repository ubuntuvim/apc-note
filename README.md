
# ape-note
一个异常简洁的笔记应用，一个让你爱不释手的笔记应用，遇上我是你的荣幸。


## 项目安装

* `git clone https://github.com/ubuntuvim/apc-note.git`
* `cd ape-note`
* `npm install`
* `bower install`

## 运行项目

* `ember serve`
* 访问[http://localhost:4200](http://localhost:4200).

### 打包项目

* `ember build` (development)
* `ember build --environment production` (production)
